# TWTree

基于vue的树形组件。

主要特色
*  支持复选框
*  可异步加载
*  拖放操作
*  右键菜单
*  按钮
*  自定义外观

[文档](https://github.com/TinyWisp/twtree/wiki/%E4%B8%AD%E6%96%87%E6%96%87%E6%A1%A3)
